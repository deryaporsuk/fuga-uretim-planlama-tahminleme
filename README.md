# Production Planning Demand Forecasting

Bu projede Fuga Mobilya üretim planlama departmanı için satın alma miktarlarının tahminlenmesine yönelik veri analizi ve makine öğrenmesi çalışmaları gerçekleştirilmiştir.

## Proje İçeriği
- SQL ile veri birleştirme ve veri hazırlama
- Qlik Sense ile veri düzenleme ve görselleştirme
- Segmentasyon yaklaşımı
- Özellik mühendisliği
- XGBoost modeli
- Zaman serisi analizi
- Holt ve ARIMA modelleri
- Tahmin performans değerlendirmeleri (RMSE, R²)

## Kullanılan Teknolojiler
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SQL
- Qlik Sense
- Minitab

## Proje Kapsamı
Çalışmada 2025 yılı üretim planlama verileri kullanılarak satın alma miktarlarının tahminlenmesi amaçlanmıştır. Farklı modelleme yöntemleri karşılaştırılmış ve en başarılı sonuçlar XGBoost modeli ile elde edilmiştir.
